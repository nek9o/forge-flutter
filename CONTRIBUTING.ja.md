# コントリビューションガイドライン

Forge Flutter Client への関心をお寄せいただきありがとうございます！本プロジェクトはアルファ版であり、皆様からのフィードバックや貢献を心よりお待ちしております。

## どのように貢献できるか

### 1. バグ報告

不具合を見つけた場合は、[GitHub Issues](https://github.com/fal-114514/forge-flutter/issues) に報告してください。報告の際は、以下の情報を含めていただけると助かります。

- 使用している Windows のバージョン
- SD-WebUI-Forge のバージョン
- 不具合の再現手順
- 可能であればスクリーンショットやログ

### 2. 機能提案

「こんな機能が欲しい」というアイデアがあれば、Issue で提案してください。軽量性を維持しつつ、利便性を向上させるアイデアを歓迎します。

### 3. プルリクエスト (PR)

コードの修正や機能追加を行う場合は、以下の手順でお願いします。

1. 本リポジトリをフォークする
2. 新しいブランチを作成する (`feature/your-feature-name` や `fix/your-bug-name`)
3. 変更をコミットする（コミットメッセージは日本語でOKです）
4. プルリクエストを送信する

### 4. ドキュメント・表現の修正

日本語や英語の表現が不自然な場所、説明が不足している場所の修正も非常に助かります。

## 開発環境のセットアップ

1. [Flutter SDK](https://docs.flutter.dev/get-started/install) をインストールします。
2. リポジトリをクローンします。
3. `flutter pub get` を実行して依存関係を解決します。
4. `flutter run -d windows` で実行します。

## ライセンス

本プロジェクトへの貢献は、すべて zlib ライセンスの下で公開されることに同意したものとみなされます。
