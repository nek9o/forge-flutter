# Contributing to Forge Flutter Client

Thank you for your interest in Forge Flutter Client! This project is currently in alpha, and we warmly welcome feedback and contributions from the community.

## How to Contribute

### 1. Bug Reports

If you find a bug, please report it via [GitHub Issues](https://github.com/fal-114514/forge-flutter/issues). When reporting, please include the following information:

- Your Windows version
- SD-WebUI-Forge version
- Steps to reproduce the issue
- Screenshots or logs, if possible

### 2. Feature Requests

If you have an idea for a new feature, feel free to open an Issue. We welcome ideas that improve usability while maintaining the app's lightweight nature.

### 3. Pull Requests (PR)

To submit code changes or new features:

1. Fork the repository
2. Create a new branch (`feature/your-feature-name` or `fix/your-bug-name`)
3. Commit your changes (commit messages in Japanese or English are both welcome)
4. Submit a pull request

### 4. Documentation & Wording Fixes

Corrections to Japanese or English wording, as well as improvements to documentation, are also greatly appreciated.

## Development Setup

1. Install the [Flutter SDK](https://docs.flutter.dev/get-started/install).
2. Clone the repository.
3. Run `flutter pub get` to resolve dependencies.
4. Run `flutter run -d windows` to launch the app.
